## 🚀 내 풀이 ⭕

딱 정석적인 투포인터로 풀면 된다

처음에 풀었던 코드가 계속 채점 되다가 틀렸다고 하는데 뭐가 틀렸는지 모르겠다..

무한루프 만들어서 안에서 직접 break 해야 풀리더라


```python
import sys
input = sys.stdin.readline
N, M = map(int, input().split())
nums = list(map(int, input().split()))

p1 = p2 = 0
total = nums[p1]
answer = 0
while 1:
    if total == M:
        answer += 1
        total -= nums[p1]
        p1 += 1
    elif total > M:
        total -= nums[p1]
        p1 += 1
    else:
        p2 += 1
        if p2 == N: break
        total += nums[p2]

print(answer)
